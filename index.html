Covid-19 Dashboard

In this project, I used a dataset containing various Covid-19 data in relation to different countries. Before making a dashboard with data taken from the dataset, I used Microsoft SQL Server Management Studio 18 to extract certain data that I wanted for the dashboard. After extracting the data that I wanted, I used Tableau Public to create an interactive dashboard that displays global death percentage, percent population infacted per country, total deaths per continent, and percent population infected. The dataset source is https://ourworldindata.org/covid-deaths.
